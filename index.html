Dropdown-angular

Simple dropdown directive for angular

Download this project as a .zip file Download this project as a tar.gz file

Dropdown-angular demo

Simple case:

<div ya-dropdown='open'>
    I am simple one level dropdown
    <button ng-click='open = !open'>click me</button>
    <div ng-if='open' ng-click='$event.stopPropagation()'>
        <ul>
            <li>First item</li>
            <li>Second item</li>
            <li>Third item</li>
        </ul>
    </div>
</div>

Nested dropdowns:

<div ya-dropdown='openParent'>
    I am nested dropdown
    <button ng-click='openParent = !openParent'>click me</button>
    <div ng-if='openParent' ng-click='$event.stopPropagation()'>
        <ul>
            <li>First item</li>
            <li>Second item</li>
            <li>
                <div ya-dropdown='openNested'>
                    I am inner dropdown
                    <button ng-click='openNested = !openNested'>click me</button>
                    <div ng-if='openNested' ng-click='$event.stopPropagation()'>
                        <ul>
                            <li>First item</li>
                            <li>Second item</li>
                            <li>Third item</li>
                        </ul>
                    </div>
                </div>
            </li>
        </ul>
    </div>
</div>
